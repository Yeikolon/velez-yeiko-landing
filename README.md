##  Descripción del Proyecto

**Derrape Labs** es una landing page profesional diseñada para un centro ficticio de simulación de conducción y deportes electrónicos. 

El sitio web presenta la identidad de la marca, sus servicios principales, instalaciones, testimonios de clientes y un formulario de contacto, con el objetivo de convertir visitantes en clientes potenciales.

La página ha sido desarrollada siguiendo un enfoque **mobile first**, garantizando una experiencia de usuario óptima en dispositivos móviles, tabletas y computadoras de escritorio.

---

##  Tecnologías Utilizadas

- **HTML5 Semántico:** Estructura clara y accesible del contenido.
- **CSS3:** Estilos personalizados con variables CSS y diseño responsive.
- **Bootstrap 5:** Sistema de grid, componentes (navbar, carrusel, formulario) y utilidades.
- **Font Awesome 6:** Biblioteca de iconos para servicios y redes sociales.
- **Git & GitHub:** Control de versiones y alojamiento del repositorio.
- **GitHub Pages:** Publicación del sitio web (si aplica).

## Comportamiento Responsive

La página se adapta a tres tamaños principales:

| Tamaño | Dispositivo | Distribución |
| :--- | :--- | :--- |
| **375px** | Móvil | Una sola columna, menú colapsable |
| **768px** | Tableta | Dos columnas en servicios y testimonios, tres en indicadores |
| **1366px** | Escritorio | Cuatro columnas en servicios, hero con tipografía más grande |
